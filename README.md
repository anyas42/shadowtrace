# 🕵️ ShadowTrace

> AI-powered cyber mystery investigation platform.

ShadowTrace is an immersive detective experience where users solve AI-generated cyber mysteries using clues, encrypted files, terminal commands, and interactive investigations.

---

## ✨ Features

- 🔐 Hacker-style terminal interface
- 🧠 AI-generated mystery cases
- 📂 Interactive evidence system
- 🎧 Audio and encrypted clues
- 🕶️ Cyberpunk-inspired UI
- 🧩 Puzzle & cipher challenges
- 🤖 AI detective assistant

---

## 🚀 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- FastAPI (Python)

### AI Integration
- OpenAI API / Gemini API
